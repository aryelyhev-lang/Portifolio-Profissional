Projetos em Destaque

Minha experiência na Amazon me deu disciplina operacional e visão de processos, habilidades que hoje aplico na organização de código e arquitetura de sistemas

API de Localidades Brasileiras

APi desenvolvida em Node.js e Express, fornece dados detalhados sobre estados e cidades brasileiras, incluindo filtros por região, sigla e capitais. O projeto destaca-se pela arquitetura modular, implementação de CORS, tratamento de erros via status codes e prontidão para deploy no Render.

Vitrine de Produtos Dinâmica

Este projeto consiste na criação de uma interface de marketplace (vitrine de produtos) totalmente funcional e dinâmica. O principal diferencial desta aplicação é a transição de um modelo estático para um modelo orientado a dados. Em vez de codificar manualmente cada card de produto no arquivo HTML, o sistema utiliza a lógica de programação para processar um banco de dados simulado (um arquivo de objetos JavaScript) e transformá-lo em elementos visuais em tempo real.

Sistema de Gestão e Processamento Escolar

Este projeto foi desenvolvido para atuar como o "motor" de processamento de um sistema escolar, utilizando Node.js. O objetivo central foi criar uma lógica de back-end altamente confiável que vai além do simples cálculo aritmético, focando na integridade dos dados e na personalização da experiência do usuário. Através de uma arquitetura modular, o sistema recebe informações complexas, valida cada entrada e entrega resultados precisos e prontos para o front-end.
